常用的都很熟悉了，但部分用起来还不是很熟练。制作一个简单的表，以便有疑问时随时查询。

| h或向左箭头 | 向左移动一个字符 |
| j或向下箭头 | 向下移动一个字符 |
| k或向上箭头 | 向上移动一个字符 |
| l或向右箭头 | 向右移动一个字符 |
| [Ctrl]+[f]  | 显示下一页 |
| [Ctrl]+[b] | 显示上一页 |
| [Ctrl]+[d] |显示下半页|
| [Ctrl]+[u]| 显示上半页|
|+|光标移动到非空格符的下一行|
|-|光标移动到非空格符的上一行|
|n<space>|光标向后移动n个字符|
|0|移动到本行最前面|
|$|移动到本行末尾|
|H|移动到屏幕最上方那行的第一个字符|
|M|移动到屏幕中间那行的第一个字符|
|L|移动到屏幕最下面那行的第一个字符|
